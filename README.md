# Smart-Biometric-Door-Lock
- Hardware
  - Finger Print sensor
  - Relay
  - esp32
  - Oled module
  - Solenoid door lock

- Software
  - C ++ Programming
  - Arduino IDE
  - Cayenne IoT cloud platform

#Project Brief:
- The door unlocks based on the fingerprint record the system has in its memory. The fingerprint(User) can be added 
   using the cloud user interface in the IoT cloud platform Cayenne
